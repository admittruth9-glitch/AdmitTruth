# Welcome to your AdmitTruth

AdmitTruth AI

> “Know Before You Choose”

Overview:

AdmitTruth AI is an AI-driven student decision intelligence system that helps users choose the right engineering path, verify college claims, and scam detection using smart analysis.



 Problem Statement:

Students often rely on misleading college information and lack proper guidance, leading to wrong career decisions and exposure to admission scams.



 Solution:

AdmitTruth AI provides:
-  Personalized career guidance and smart carrer councelling 
-  Real vs Brochure college verification  
- AI-based scam detection  



Features:

-  AI-based student path recommendation  
-  Smart Counselling Path System (question-based guidance)  
- College comparison (Reality vs Marketing data)  
-  Personalized decision engine  
-  Scam detection with risk scoring  
-  College suggestions based on rank, interest, and goals  



How It Works:

1. Student enters details (rank, interest, preferences)  
2. AI asks guided questions for deeper analysis  
3. System generates multiple career paths  
4. Each path provides detailed insights and roadmap  
5. User selects a path → gets next steps + college suggestions  
6. College data is verified with reality insights  
7. Scam analyzer checks suspicious admission messages  



  Tech Stack:

•Frontend: React + Tailwind (TSX/.CSS)
•Logic ("Backend"): Embedded TypeScript (in src/lib)
•Database: Static TypeScript Data (in src/data)
•DevOps: Vite + Playwright + Vites
•AI Logic: Rule-based system (extendable with APIs)
